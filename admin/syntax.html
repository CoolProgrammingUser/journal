Coding syntax

This helps me format my entries.

Classes

Class Description
p# Identifies a person in the person list. Can still be recognized as plain text (not as a class). Specific names can be chosen with data-use. Creates a link to a search of that person. "p0" designates an unknown person.
t# Identifies a topic in the topic list. "t0" designates an unclassified topic.
h# Identifies honesty. "h0" designates an unclassified form of truth.
e# Identifies emotions. "e0" designates an unclassified emotion.
elaborate Allows triggering the appearance of asides by clicking the contained text. The aside to be triggered immediately follows this element.

Data attributes

Attribute Function
use Which of a person's names to use. Using "extra" followed by a number makes it possible to access the extraNames array (1-based index).
heading The heading of an aside.
importance How important something is. Currently 0-3. 0 is implied if importance isn't indicated.
date The date of the element's contents. Format = year/month/day.
currency The currency of a value.
untranslatable? Something that can't be translated without corrupting the original message. Possibly set equal to a basic description of the problem.
revisit Something that need to be revisited. Preferably set equal to a basic description of the problem.
