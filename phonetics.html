Phonetics

This describes how to interpret my phonetic code. I use this code any time I want to unambiguously describe how to pronounce something, so if you don't know how to use it, my clarifications won't do you much good (although I did try to make it as straightforward as possible). I capitalize the vowel of the syllable that has emphasis. Also, pronunciations are subject to how I think they should be pronounced. (e.g. You can just get over it if you think the word "envelope" should be pronounced "Ahnvuhlop" instead of "Envuhlop".)
